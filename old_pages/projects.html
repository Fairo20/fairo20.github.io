Projects

VMA: Virtual Memory Architecture
This project is my current topic of research, optimized data structures for C++ architectures. The finished product is a fully portable library of unique data structures with different uses, all of which have a significant speedup over their competitors, as well as other advantages. The paper has been accepted and can be seen above, and is in the process of being published through the TAMU OAKTrust Digital Repository. The GitHub repository will be made available soon.

CS435: GPU Merge Sort
This project was completed as the final project of CS435, Parallel Computing, in which I created a GPU Merge Sort algorithm in CUDA and compared it to a standard parallel merge sort, as well as other members' different sorting algorithms. This was ran on the Texas A&M Grace computing cluster using Lawrence-Livermore National Laboratory's Caliper and Thicket data collection systems for GPU and CPU parallelized timing.

Fujitsu R&D Networking Robot
NOTE: Code is proprietary, so these are the prototype video and picture from my final intern presentation
As shown, I developed a prototype wireless robot utilizing an STM32 microcontroller running Micropython, L298N motor controllers, an ADXL345 accelerometer, and an Orange Pi 3 running a minimal Debian installation for wireless communication. This prototype was intended to be a starting point for a design concept to navigate a server room autonomously to perform routine maintenance. Other topics of my internship included the development of a vision dataset for Fujitsu servers, setting up new Py boards for the development team, and developing a patent proposal for a server rack attachment mechanism.

CSCE 315 Project: Smook
NOTE: Due to operational costs, the frontend is still hosted here, however the backend is not currently hosted so items will not correctly populate. A demo can be provided upon further inquiry.
This project is a fullstack web application designed to create a smoother ordering system for Smoothie King. I was in charge of the backend and hosting solutions, which were created using Java Spring Boot, with the backend hosted on Google Cloud and the frontend hosted on Firebase. The frontend of the system was created primarily through Vue.js, and communication between the two was handled via the Spring API and the Vue shop manager. This project was created using the Agile system, and I can provide the documentation for this upon further contact.

CSCE310 Project: Love Master
This project was completed in 2 weeks in Fall 2022, which involved creating an HTML-JS front-end, a connected MySQL database, and a Java back-end created with Spring Boot to allow for requests to and from the front-end and the database. The project allows for a "dating experience" between users, and features automatic scheduling, User and Admin priviledges, triggered titles, and reviews for scheduled dates.

Infinite Recharge Score Calculator
This project is a fully front-end webapp designed to calculate the score of a match of the 2020 FIRST Robotics Competition game, Infinite Recharge. The main feature is the queue, which enables both the undo and redo functions. This allows for users to take back incorrect inputs and redo accidental undo statements.

HowdyHack 2021 Project: VTuber Soundboard
This project was completed in 8 hours for HowdyHack 2021, Texas A&M's annual fall hackathon. It utilizes a similar queue system to the Score Calculator detailed above, however it stores music files instead of numbers. This soundboard also utilizes the Howl.js library in order to adjust the volume and speed of the assets.

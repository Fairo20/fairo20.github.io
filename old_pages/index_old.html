CSCE 315 Project: Smook
NOTE: Due to operational costs, the frontend is still hosted at https://smook-app.web.app, however the backend is not currently hosted so items will not correctly populate. A demo can be provided upon further inquiry.\nThis project is a fullstack web application designed to create a smoother ordering system for Smoothie King. I was in charge of the backend and hosting solutions, which were created using Java Spring Boot, with the backend hosted on Google Cloud and the frontend hosted on Firebase. The frontend of the system was created primarily through Vue.js, and communication between the two was handled via the Spring API and the Vue shop manager. This project was created using the Agile system, and I can provide the documentation for this upon further contact.

CSCE310 Project: Love Master
This project was completed in 2 weeks in Fall 2022, which involved creating an HTML-JS front-end, a connected MySQL database, and a Java back-end created with Spring Boot to allow for requests to and from the front-end and the database. The project allows for a "dating experience" between users, and features automatic scheduling, User and Admin priviledges, triggered titles, and reviews for scheduled dates.

Infinite Recharge Score Calculator
This project is a fully front-end webapp designed to calculate the score of a match of the 2020 FIRST Robotics Competition game, Infinite Recharge. The main feature is the queue, which enables both the undo and redo functions. This allows for users to take back incorrect inputs and redo accidental undo statements.

HowdyHack 2021 Project: VTuber Soundboard
This project was completed in 8 hours for HowdyHack 2021, Texas A&M's annual fall hackathon. It utilizes a similar queue system to the Score Calculator detailed above, however it stores music files instead of numbers. This soundboard also utilizes the Howl.js library in order to adjust the volume and speed of the assets.
